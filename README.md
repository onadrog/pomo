# POMO

Simple "pomodoro" thing written in C using [raylib](https://github.com/raysan5/raylib/)


## Build

 ```sh

$ ./build.sh
$ ./pomo

```
## Keybinds

| Key     | Action       |
|---------|--------------|
| `space` | toggle pause |
